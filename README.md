# bioXL3p

bioXL3p is an Excel-based simulation tool for teaching and learning activated sludge processes, including carbon, nitrogen, and phosphorus removal.  

The tool is designed for **educational use**, supporting simulation-based learning and scenario-driven teaching in environmental engineering curricula.

> **Note:** Professional design or operational decision-making is **not permitted** with this version. The author does not guarantee the accuracy of results for real-world applications.

## üõ† Installation & Setup

bioXL3p is distributed as an Excel add-in (`.xlam`). No installer required.

### Requirements
- Microsoft Excel (Windows required)
- Macros enabled
- VBA access permitted (first-use prompt)

### Installation Steps
1. Download `bioXL3p.xlam` from this repository.
2. Open **Excel**.
3. Navigate to: **File ‚Üí Options ‚Üí Add-ins**
4. From the bottom dropdown, select **Excel Add-ins** ‚Üí **Go**
5. Click **Browse**, choose `bioXL3p.xlam`, confirm
6. Ensure the add-in is checked
7. Restart Excel

### Using the Tool
After installation, a **bioXL3p** tab will appear in Excel.

From this tab you can:
- Define influent characteristics
- Configure reactors and flowsheets
- Run biological nutrient removal simulations
- View structured model outputs via a custom TreeView UI
